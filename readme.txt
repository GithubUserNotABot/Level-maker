Level editor (kinda)!!

Wasd to control the character

3 to save your level
4 to load the levels you have made

5 to merge the levels
6 to create blank level files
7 to delete levels

1 to create a beginning in a level
2 to create an end in a level

Left click to make an obstacle
right click to move your character to mouse position

Left Control to backtrack a block you made
